# Лабораторная работа номер 4 по предмету: "Программирование на языке python"

## Введение
Цель работы: Научиться реализовывать приложения с асинхронной моделью управления  
Постановка задачи: Реализовать асинхронный исполнитель задач, обрабатывающий задачи из
очереди с использованием расширяемых обработчиков.

## Структура проекта
В папке src лежат основные файлы, которые содержат в себе:  
1-"12.txt", содержит данные для генерации задач  
2-"init.py"  
3-"contract.py", содержит класс, имеющий нужный метод для проверки, содержит ли в себе класс метод генерации  
4-"descriptors.py", содержит в себе дескрипторы  
5-"exceptions.py", содержит в себе пользовательские ошибки  
6-"executor.py", содержит в себе испольнитель задач  
7-"generator_1.py", содержит в себе генератор задач  
8-"generator_2.py", содержит в себе генератор задач  
9-"generator_3.py", содержит в себе генератор задач  
10-"handlers.py", содержит в себе worker  
11-"logg.py", содержит в себе настройки для логгирования  
12-"logger.py", сам logger  
13-"main.py", точка входа, содержит в себе демонстрацию работы проекта  
14-"models.py", содержит в себе модель задачи  
15-"proc.py", содержит в себе функцию проверки на наличие метода генерации при помощи isinstance, которая также собирает все задачи в один список  
16-"protocol.py", содержит в себе класс для проверки класса на наличие worker'а  
17-"TaskQueue.py", содержит в себе асинхронную очередь  
18-папка tests, содержит в себе тесты и необходимые для них данные
